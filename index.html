bienvenue sur la page du userscript OutOfAmazon - achetez vos livres localement

Voir sur GitLab Voir sur Github Installer

Qu'est ce que c'est que ce bazar?

OutOfAmazon est un petit programme qui vous permettra lorsque vous cherchez un livre de le retrouver sur le site de votre libraire local. Ce qui vous permettra de lui commander, voire de lui rendre une petite visite pour aller cherchez votre livre. Et de le nourrir lui plutôt que l'ogre. Pour l'instant à la version 0, votre libraire c'est Stéphane ;-) mais il est facile de mettre le votre en bricolant un peu...

Comment je l'installe?

Il vous faut deux choses :

  1. Une extension installée sur un navigateur moderne (cliquez sur les liens suivant votre cas)
    • Greasemonkey sur Firefox,
    • ou bien Tampermonkey sur Chrome.
  2. le script utilisateur outOfAmazon.user.js

Ca va mieux en le disant

Ce projet n'a rien à voir avec les boutiques concernées. Pas celle de l'ogre bien sûr, mais pas non plus celle de Stéphane, même si c'est au cours d'une discussion avec lui que l'idée est née...
